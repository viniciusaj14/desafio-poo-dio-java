# Desafio POO (DIO)

Implementação do desafio de **Programação Orientada a Objetos** aplicando:
- Abstração
- Encapsulamento
- Herança
- Polimorfismo

## Estrutura do projeto
- `src/br/com/dio/desafio/dominio`
  - `Conteudo`
  - `Curso`
  - `Mentoria`
  - `Dev`
  - `Bootcamp`
- `Main.java`: ponto de entrada para simular inscrição/progresso e cálculo de XP.

## Como executar
1. Abra o projeto em uma IDE (IntelliJ/Eclipse/VS Code).
2. Execute a classe `Main.java` com JDK 17+.
3. Veja no console os conteúdos concluídos e o XP acumulado.

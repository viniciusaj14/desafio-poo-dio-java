# Desafio POO (DIO)

Projeto implementado do zero para praticar os pilares de Programacao Orientada a Objetos (POO) em Java.

## Pilares aplicados
- Abstracao (`Conteudo` como base para `Curso` e `Mentoria`)
- Encapsulamento (atributos privados + getters/setters)
- Heranca (`Curso`/`Mentoria` estendem `Conteudo`)
- Polimorfismo (chamada de `calcularXp()` via tipo `Conteudo`)

## Estrutura do projeto
```text
src/
  Main.java
  br/com/dio/desafio/dominio/
    Conteudo.java
    Curso.java
    Mentoria.java
    Bootcamp.java
    Dev.java
```

## Tecnologias
- Java 17+
- GitHub + Git

## Como executar
- IDE: execute `Main.java`
- Terminal: compile/execute a partir de `src/` (ou rode via IDE)

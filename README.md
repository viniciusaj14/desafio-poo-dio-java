# Desafio POO (DIO)

Implementação do desafio de **Programação Orientada a Objetos** aplicando:

- Abstração
- Encapsulamento
- Herança
- Polimorfismo

## Estrutura do projeto

```
src/
├── Main.java
└── br/com/dio/desafio/dominio/
    ├── Conteudo.java
    ├── Curso.java
    ├── Mentoria.java
    ├── Bootcamp.java
    └── Dev.java
```

## Como executar

**IDE (recomendado)**
1. Abra na sua IDE (IntelliJ/Eclipse/VS Code com extensão Java).
2. Rode `Main.java` com JDK 17+.

**Terminal**
```bash
javac -d out src/br/com/dio/desafio/dominio/*.java src/Main.java
java -cp out Main
```
